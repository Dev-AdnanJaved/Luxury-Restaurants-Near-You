# Location Tracker

## Replit Deploy Steps

1. Create a Node.js Repl on replit.com
2. Upload these files (keep folder structure)
3. Go to Secrets tab and add:
   - BOT_TOKEN = your telegram bot token
   - CHAT_ID   = your telegram chat id
4. Hit Run ▶️

## Bot Commands
- /locations    → Last 5 locations
- /locations 10 → Last N locations  
- /last         → Most recent detection
- /health       → Uptime & stats
- /help         → Menu

## Keep Alive (Free Tier)
Add UptimeRobot monitor → your-repl-url.repl.co/ping → every 5 min
